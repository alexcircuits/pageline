Setup Guide

  1. Go to discord.com/developers/applications
  2. Click New Application and name it anything
  3. Copy the Application ID from the General Information tab
  4. Paste the ID above, click Save, then Connect to Discord
  5. For book cover artwork in Discord, upload images named book_cover and reading under Rich Presence → Art Assets
